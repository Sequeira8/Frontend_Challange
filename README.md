# Frontend_Challange
DU:RO - Frontend Challange

## Setup

### Install:
- npm
- Node.js
- gulp
- sass

### Run inside project folder:
```
npm run execute
```
Installs all dependencies and modules and opens browser with module.

## Other npm scripts

### Install dependencies and modules:
```
npm run install
```

### Run gulp:
```
npm run gulp
```
